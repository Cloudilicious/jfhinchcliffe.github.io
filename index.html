Projects

patstats

I created this app to warn our team if something was wrong with real time information on the network... and added the weather info so we could see if a jacket was required when getting lunch :)

Features

  • Devise for authentication
  • Uses PTV API for real-time public transport information
  • Forecast.io gem to get weather conditions
  • Allows user to create / update / destroy checked stops.
  • Checks selected stops, and returns the percentage responding with real time data
  • Outputs real time status and weather to the team Geckoboard via JSON

fantasy_football_app

This was created in week 3 of my Coder Factory Academy course. The app allows you to create teams, assign them stats (attack, defense etc.) and then simulate a game between teams.

Features

  • Uses random numbers to simulate games
  • Reads and writes from CSV
  • Separate Game and Team classes
  • Rudimentary testing with RSpec

ndis_plus

Proof of concept for a Code for Australia group project. Displays NDIS providers on a searchable map and was intended to illustrate how users and providers would be better serviced with richer data.

Features

  • Devise for authentication
  • GMaps4rails gem for geocoding
  • Outputs to Google Maps
